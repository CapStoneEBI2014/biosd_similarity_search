-- Application building and running
- Java JDK 1.8.x, 1.7.x, http://www.oracle.com/technetwork/java/javase/downloads/index.html
- Jena 2.11.1, http://jena.apache.org/download/index.cgi
 	

-- Development Environment
- Eclipse, java-kepler-SR1, IDE for Java developers (later versions should be fine), http://www.eclipse.org/downloads/
    - This includes EGit (https://www.eclipse.org/egit/), which is used to work with Git/GitHub
    
- Optionally, Git can be used outside Eclipse
  - As a command line tool (http://git-scm.com/downloads)
  - As a graphical interface, e.g., http://www.syntevo.com/smartgithg
